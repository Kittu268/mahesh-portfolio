A modern, high-performance personal portfolio website built with the **MERN** mindset (React + Vite) and deployed via **Vercel**. 
It features real-time GitHub integration, glassmorphism UI, and scroll-driven animations.

## ?? Live Demo
**[https://mahesh-portfolio-peach.vercel.app](https://mahesh-portfolio-peach.vercel.app)**

## ??? Tech Stack
- **Frontend:** React.js, CSS3 (Glassmorphism), Vite
- **Deployment:** Vercel CI/CD
- **APIs:** GitHub REST API (for fetching repo stats)
- **Forms:** FormSubmit.co (Email forwarding)

## ?? Installation & Setup

If you want to run this project locally, follow these steps:

### 1. Clone the repository
\\\ash
git clone https://github.com/Kittu268/mahesh-portfolio.git
cd mahesh-portfolio
\\\

### 2. Install Dependencies
\\\ash
npm install
\\\

### 3. Run Development Server
\\\ash
npm run dev
\\\
Open [http://localhost:5173](http://localhost:5173) to view it in the browser.

## ?? Deployment (Vercel)

1. Install Vercel CLI: \
pm i -g vercel\
2. Run deployment command:
   \\\ash
   vercel --prod
